Before running the application you should do this steps : 

1 : Make sure you add .JSON in your system path. the file can be found in the project directory.

2 : Go to the project server  directory (FB_Messenger_bot/server/.env) and assign your path to GOOGLE_APPLICATION_CREDENTIALS
    change <your json path> to your .json file path. don't forget to add it your system path and restart your terminal
